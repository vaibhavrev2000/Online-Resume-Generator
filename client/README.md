## Frontend ⚛️

Following are the steps to run the frontend of the portgen on your local environment. All the frontend code will go in the `client` directory. 

1. Navigate to `client` folder.
  ```sh
  cd client
  ```
2. Install all the required packages and dependencies.
  ```node
  npm install
  ```
3. Run the server.
  ```node
  npm start
  ```
4. [Click here](http://localhost:3000) to view it in the browser OR navigate to
  ```text
  http://localhost:3000
  ```
